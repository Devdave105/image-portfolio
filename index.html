How I Built This

Every part of this portfolio was handcrafted with creativity and code. Here’s how it came to life:

HTML Structure

I started by designing the structure of each section with semantic HTML — the navbar, hero, and gallery. Everything was built from scratch to ensure clean, reusable code.

CSS Styling

Using modern CSS, I focused on clean layouts, color balance, and responsive design. I used grid and flexbox for smooth alignment and a sleek visual flow across all screens.

⚡

JavaScript Interactions

Finally, I added interactivity with JavaScript — from the image uploader to scroll-triggered animations and the hamburger menu. Every animation makes the experience more alive.
